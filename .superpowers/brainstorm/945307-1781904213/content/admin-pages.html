Which admin pages to build first?

All are listed in the spec. Pick your priority.

1

Dashboard — KPI Cards & Charts

Asset KPIs: total assets, active WOs, overdue maintenance, warranty expiring. Charts: assets by status/category, maintenance trends, cost trends.

2

Work Orders Tab

Kanban board (pending → in_progress → completed) + list view toggle. Filters: status, asset, contractor, type, date range.

3

Maintenance Tab

Calendar view of upcoming schedules, overdue alerts, schedule management (create/edit/complete).

4

Import / Export

CSV upload with column mapping UI, template download, export assets / work orders / costs.

5

QR Batch Generation

Select assets → generate printable PDF with multiple QR code labels.
