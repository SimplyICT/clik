Work Orders Tab — View Preference

The tab goes inside Asset Management, alongside Assets / Parts / Custom Fields / Audit Log

Default view when opening the tab?

A

List view (default)

Table with filters at the top. Toggle button to switch to Kanban. Clean, scannable.

B

Kanban board (default)

Columns: Pending → In Progress → Completed. Drag/tap to change status. Toggle to list view.

C

Show both side-by-side

Kanban on top, list below, or split the page. See the big picture and details at once.
