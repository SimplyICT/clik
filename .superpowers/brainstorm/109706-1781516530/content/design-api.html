Design: API Endpoints

All under /api/asset-management/*, auth required

Assets

MethodEndpointDescription
GET/assetsList (with filters: category, status, customer, contractor)
GET/assets/{id}Detail with service history
POST/assetsCreate (generates QR code)
PATCH/assets/{id}Update
POST/assets/{id}/retireRetire (manager only)
POST/assets/{id}/transferTransfer customer (manager only)

Jobs (links asset to existing request)

MethodEndpointDescription
POST/assets/{id}/create-jobCreate request linked to asset, type = install/move/retire/inspect/repair/transfer
GET/assets/{id}/jobsList all jobs for this asset

Parts

MethodEndpointDescription
GET/POST/PATCH/DELETE/partsCRUD
POST/parts/{id}/record-usageRecord parts used during a job

QR Codes

GET/assets/{id}/qrReturns QR code image (PNG)
