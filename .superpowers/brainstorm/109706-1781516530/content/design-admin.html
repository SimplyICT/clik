Design: Admin Portal — Asset Management Hub

New nav item: "Asset Management" — replaces current "Assets"

Tabs within the page:

TabContent
All AssetsFull table with search, filters (category, status, customer, contractor), batch actions. Columns include QR code preview.
Asset DetailFull detail view with all fields, photo gallery, parts list, service history timeline. Action buttons: Create Job, Edit, Retire, Transfer, Print QR.
Parts InventoryManage parts catalog. Add/edit parts, set thresholds, view usage history. Low-stock warnings.
Custom FieldsManager-only: Define which custom fields exist per category
Audit LogManager-only: All asset lifecycle events (created, edited, retired, transferred, jobs created)

Permission differences:

  • Contractor: Can create/edit/retire assets, create jobs, record parts usage
  • Manager: Everything above + permanent delete, transfer ownership, view audit log, manage custom field definitions
  • Customer: View their own assets + add notes, request service
